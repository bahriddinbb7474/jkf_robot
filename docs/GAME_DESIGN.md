# Game Design

## Concept and audience

`JKF_robot` is a local browser-based 2D top-down robot combat game for teenagers. Sessions should be short, controls understandable, and progression visible without a large campaign.

## Main loop

`create player -> battle -> victory/defeat -> reward -> shop -> garage -> new mission`

## MVP mechanics

- WASD movement and mouse aiming/actions.
- Laser, rocket, and sword weapons.
- Basic, fast, and shooter enemies plus one boss.
- Local player progress, missions, rewards, permanent purchases, shop, and garage.
- Optional math/English multiple-choice bonus questions.
- Basic sound effects and volume setting.

## Excluded features

No multiplayer, online services, accounts, admin panel, AI question generation, repair/energy systems, mobile controls, complex layered robot visuals, or large campaign.

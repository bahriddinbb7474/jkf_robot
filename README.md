# JKF_robot

Local browser-based 2D top-down robot game. The repository is currently limited to the Stage 0-A project foundation; gameplay is not implemented.

## Requirements

- Node.js 20 or newer
- npm

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
npm run format
npm run preview
```

Open the URL printed by Vite. The expected screen contains `JKF_robot`, `MVP Foundation Ready`, and `Stage 0-A`.

Project rules are in `AGENTS.md`; current scope and plans are in `docs/`.
